# Lab 8: Timers

What to expect on each source file:

Excercise 1:
- Turning a LED 60ms on and 60ms off by using the Timer0 module.

Excercise 2:
- Turning a LED on and off the maximum time possible by using the Timer0 module.

Excercise 3:
- Usage of the T0CKI pin and the Timer0 to turn a LED on after pressing 10 times a push button. Then repeat the same to turn it off.

Excercise 4:
- Measuring the ammount of push button-pressings during 10 seconds and showing the result multplying it 6 times.
